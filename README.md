# tg-assistance-bot

Telegram-бот для поддержки и поднятия настроения: отправляет комплименты по нажатию кнопки.

## Как пользоваться

1. Откройте диалог с ботом в Telegram.
2. Нажмите команду `/start`.
3. Пользуйтесь кнопками:
   - **Старт** — бот кратко подскажет, что он умеет.
   - **Комплимент** — бот отправит случайный комплимент.

## Если бот не отвечает

- Проверьте, что вы нажимаете кнопки после `/start`.
- Если вы отправляете произвольный текст (не через кнопки), бот подскажет, что отвечает только на нажатие кнопок.
